# Image Dimensions Specifications for Arabic Global School Website

This document provides recommended image dimensions for all images used across the website. Dimensions are optimized for responsive design and retina displays.

## General Guidelines
- **Format**: JPEG for photos, PNG for graphics with transparency
- **Quality**: High quality (85-95% for JPEG)
- **Retina Support**: Provide 2x versions for critical images (hero images, main content)
- **Aspect Ratios**: Maintain specified ratios to prevent distortion
- **File Size**: Optimize for web (aim for <500KB per image when possible)

## NOTES FOR CLIENT

1. **Retina Displays**: All dimensions listed are 2x for retina/high-DPI displays. The website will automatically serve appropriate sizes.

2. **Responsive Images**: Some images (About Us Vision/Mission) use the same file for all breakpoints. The website handles responsive sizing via CSS.

3. **Aspect Ratios**: Maintaining exact aspect ratios is critical to prevent distortion. Use the specified ratios as guidelines.

4. **File Naming**: Maintain exact file names including Arabic characters and spaces. The website references these exact paths.

5. **Optimization**: While high quality is important, ensure images are optimized for web delivery to maintain fast page load times.

6. **Video Format**: The meal section video should be MP4 (H.264) at 1920 × 1080px minimum, 16:9 aspect ratio.
